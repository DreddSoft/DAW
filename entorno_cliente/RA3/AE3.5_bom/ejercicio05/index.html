Ejercicio 5. OBJETO DOCUMENT

El DOM permite a los desarrolladores ver una página HTML de otra manera, como un conjunto de objetos que JavaScript puede consultar y modificar.

  • Crea un nuevo documento HTML con una imágen, un formulario con dos campos para introducir tu nombre y apellidos y 4 párrafos.
  • El contenido de los párrafos será "Párrafo N", donde N hace referencia al número de párrafo
  • Recupera el segundo párrafo utilizando los 4 métodos y muestralos por consola seguido del texto "obteniendo mediante METODO" donde METODO hace referencia al método utilizado para la obteneción de contenido.
