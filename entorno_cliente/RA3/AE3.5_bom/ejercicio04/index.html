Ejercicio 4. OBJETO LOCATION

Implementa una página web que tenga un texto en el encabezado principal "¿Desea contactar con nosotros?".

El siguiente texto en un párrafo "Haz click en el siguiente enlace para <abrir un Webmail aleatorio>, siendo el texto contenido entre <> un enlace."

Cuando se presione dicho enlace, se generará un valor aleatorio entre 0 y 2.

  • Si el valor generado es 0, la aplicación redirigirá a la página web de login de hotmail.
  • Si el valor es 1, la aplicación redirigirá a la página web de login de Gmail.
  • Si el valor es 2, la aplicación redirigirá a la página web de login de serviciodecorreo.es
